## ইনডেক্স আর্কিটেকচার

**ইনভার্টেড ইনডেক্স** — সিম্পল স্ট্রিং-স্ক্যান নয়; প্রতি কীস্ট্রোকে ৬,২৩৬ আয়াত কখনো স্ক্যান হয় না।

- **৩টি ফিল্ড-ইনডেক্স:** আরবি (নরমালাইজড: হরকত-মুক্ত, আলিফ/হামজা-রূপ একীভূত) · বাংলা · ইংরেজি — token → verse-id পোস্টিং লিস্ট
- **ভোকাবুলারি:** আরবি ১৪,৬৫৮ / বাংলা ১১,২৭৭ / ইংরেজি ৬,২৭৩ টোকেন; sorted-array + বাইনারি-সার্চে প্রিফিক্স-রেঞ্জ লুকআপ
- **স্ট্রাকচারাল লুকআপ:** meta টেবিল [সূরা, আয়াত, জুয, পৃষ্ঠা, রুকু, হিযব] — রেফারেন্স (2:255), জুয/পৃষ্ঠা/রুকু কোয়েরি সরাসরি
- **এক্সটেনসিবল খাম:** transliteration/root/topic/tafsir = একই খামে নতুন `{tokens, post}` ব্লক — কোয়েরি-রাউটার বাড়ালেই হবে (AI/semantic সার্চের entry-point `searchByTopic` স্টাব রাখা)
- **একবার তৈরি, চিরদিন ক্যাশড:** idle-চাংকে ব্যাকগ্রাউন্ড বিল্ড (UI-ব্লক নেই) → Cache API-তে সংরক্ষিত → পরের সেশনে এক-পার্স লোড; সম্পূর্ণ অফলাইন

## সার্চ-ক্ষমতা

তাৎক্ষণিক (২০০ms ডিবাউন্স) · প্রিফিক্স · কনটেইনস · **টাইপো-টলারেন্স** (edit-distance-1) · **"কোটেড ফ্রেজ"** (আরবি/বাংলা substring-verify) · ম্যাচ-হাইলাইট (`<mark>`) · সাম্প্রতিক+জনপ্রিয় সার্চ (RS-স্কিমায়) · রেফারেন্স (2:255, বাংলা সংখ্যাও) · জুয/পৃষ্ঠা/রুকু · সূরা-নাম/নম্বর (আগের আচরণ অক্ষত)

**ফলাফল-কার্ড:** সূরা-নাম + আয়াত + জুয/পৃষ্ঠা/রুকু চিপ + হাইলাইটেড প্রিভিউ + 🔖❤️🖍️📝 স্ট্যাটাস → ট্যাপে সরাসরি আয়াতে (রিডার-রিস্টোর + টিল-রিং ফ্ল্যাশ + রিডিং-স্টেট অক্ষত)
**ফিল্টার:** সব / মক্কী / মাদানী / 🔖 / ❤️ / 🖍️ / 📝 · **সর্ট:** প্রাসঙ্গিকতা / কুরআন-ক্রম / সাম্প্রতিক-পঠিত / বুকমার্ক-আগে
**কীবোর্ড:** ↑↓ নেভিগেশন + Enter; ফলাফল-সংখ্যা aria-live ঘোষণা

## Developer API (`window.searchEngine`)

```js
await searchEngine.search(q, {type, juz, sort})  // → {hits:[{id,sc}], field, words, ms}
searchEngine.searchArabic(q)      searchEngine.searchTranslation(q)
searchEngine.searchByPage(604)    searchEngine.searchByJuz(30)
searchEngine.searchByTopic(q)     // future-ready stub
searchEngine.meta[id]             // [surah, ayah, juz, page, ruku, hizb]
```
