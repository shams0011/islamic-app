# READER_ENGINE.md — কুরআন রিডিং ইঞ্জিন (P0.5, v162)

## স্টেট স্কিমা (একক, এক্সটেনসিবল)

localStorage কী: `profileKey('quran_state_v1')` — **রিডিং-সংক্রান্ত আর কোনো আলাদা কী নেই।** ম্যানেজার: `RS` (load / save-debounced / verse / patchVerse / readCount)।

```js
{
  session: { s, a, juz, hizb, rub, ruku, page, ts },        // চলমান পাঠ-প্রসঙ্গ, স্ক্রলে অটো-আপডেট
  verses: {                                                  // sparse: শুধু চিহ্নিত আয়াত
    "2:255": { b, f, hl(1-4), note, noteTs, lastRead, count, audio, mem, rev, tags }
  },                                                         // খালি/০ মান স্বয়ংক্রিয় মুছে যায়; ভবিষ্যৎ ফিল্ড (AI ট্যাগ,
                                                             // memorization, cloud-sync meta) মাইগ্রেশন ছাড়াই মার্জ হয়
  stats: { daily: {"YYYY-MM-DD": seconds}, lifetime: seconds },
  settings: { font, size, trSize, lh, tajweed, showTr, showNum, hc, autoScroll }
}
```

## মাইগ্রেশন নোট

- পুরনো কীগুলো (`quran_pos`, `quran_font`, `quran_size`) প্রথম লোডে একবার স্কিমায় ইমপোর্ট হয়ে **মুছে যায়** — ব্যবহারকারীর অবস্থান/ফন্ট-পছন্দ অক্ষত।
- ডেটাসেটে `page` (১-৬০৪) যোগ হয়েছে (P0.2-এর মতোই ক্যানোনিকাল-উৎস); যোগের পর ৬,২৩৬ আয়াত পুনঃযাচাই — টেক্সট ও মেটাডেটা ০ অমিল।
- বিসমিল্লাহ: ডেটাসেটের আয়াত-১-এ এমবেডেড (দেখুন QURAN_VERIFICATION.md) — রিডার এখন রেন্ডার-সময়ে স্ট্রিপ করে আলাদা মুসহাফ-প্যানেল হেডারে দেখায় (সূরা ১ ও ৯ ব্যতীত); ডেটা অপরিবর্তিত।

## রিডার UX

- **হেডার:** সূরার বাংলা+আরবি নাম, মক্কী/মাদানী, আয়াত, জুয, হিযব, রুকু, পৃষ্ঠা-রেঞ্জ, সিজদা-চিপ + ক্রিম বিসমিল্লাহ-প্যানেল।
- **ফুটার (ভাসমান):** সূরা-প্রগ্রেস বার + % + আনুমানিক বাকি সময় + দ্রুত-বুকমার্ক + তালিকা + সেটিংস + শীর্ষে-যান। শুধু রিডার খোলা থাকলে দৃশ্যমান।
- **আয়াত-কার্ড অ্যাকশন:** 🔖 বুকমার্ক · ❤️ প্রিয় · 🖍️ হাইলাইট (৪ রঙ সাইকেল, ৫ম ট্যাপে মুছে) · 📝 নোট (এডিট/ডিলিট, টাইমস্ট্যাম্প) · 📋 কপি · ↗️ শেয়ার — প্রতিটি অ্যাকশন `refreshVerse(n)` দিয়ে **শুধু সেই কার্ড** রি-রেন্ডার করে (মাপা: পুরো রি-রেন্ডারও মাত্র ৫ms, সিঙ্গেল-কার্ড <১ms)।
- **বুকমার্ক-ম্যানেজার শিট:** ফিল্টার (সব/🔖/❤️/🖍️/📝) + টেক্সট-সার্চ + সর্ট (সাম্প্রতিক/মুসহাফ-ক্রম) + ট্যাপে সরাসরি আয়াতে জাম্প + ✕ মুছুন + 📤 সব নোট এক্সপোর্ট।
- **সেটিংস শিট:** ৩ ফন্ট, আরবি সাইজ, অনুবাদ সাইজ, লাইন-স্পেসিং, তাজবীদ রঙ অন/অফ, অনুবাদ, আয়াত-নম্বর, উচ্চ-কনট্রাস্ট (সাদা পাতা), অটো-স্ক্রল (P0.6-এ কার্যকর হবে), শব্দে-শব্দে।
- **সেশন:** স্ক্রলেই সূরা/আয়াত/জুয/হিযব/রুব/রুকু/পৃষ্ঠা সংরক্ষিত; রিস্টার্টে অটো-রিজিউম; সক্রিয়-পাঠ সময় (১৫s টিক, ট্যাব-দৃশ্যমান + ২ মিনিটের মধ্যে কার্যকলাপ থাকলে) দৈনিক+লাইফটাইম পরিসংখ্যানে জমে — P0.8-এর পরিসংখ্যান-ড্যাশবোর্ডের ভিত্তি।
